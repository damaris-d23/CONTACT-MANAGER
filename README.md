# Contact Manager

## Overview

Contact Manager is a Java application designed to help users organize and manage personal contacts efficiently. The application provides a simple and intuitive way to store, search, update, and remove contact information while demonstrating object-oriented programming principles and data management techniques.

This project was developed as an academic project to practice Java programming, data structures, file handling, and CRUD (Create, Read, Update, Delete) operations.

## Features

* Add new contacts
* View all saved contacts
* Search contacts by name
* Update existing contact information
* Delete contacts
* Store contact details such as name, phone number, and email address
* Simple and user-friendly interface
* Contact management through CRUD operations

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collections Framework
* File Handling
* IntelliJ IDEA

## Main Functionalities

### Add Contact

Users can create and save new contacts by providing personal information such as:

* Full name
* Phone number
* Email address

### View Contacts

Displays all stored contacts in an organized format.

### Search Contacts

Allows users to quickly find contacts using their name.

### Update Contact

Existing contact information can be modified whenever necessary.

### Delete Contact

Contacts can be permanently removed from the contact list.

## Installation

### Clone the Repository

```bash
git clone https://github.com/damaris-d23/CONTACT-MANAGER.git
```

### Open the Project

Open the project using IntelliJ IDEA or any Java IDE.

### Build the Project

```bash
mvn clean install
```

## Running the Application

Run the main class:

```text
Main.java
```

The application will start and display the available contact management options.
